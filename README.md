
# Todo
Basic todo project for teaching purpose only 

# Requirements
<ul>
 <li>maven comfigured to be able to run mvn, or use wrapper</li>
  <li> Java 17</li>
</ul>

# Run unit tests
```sh
mvn test -Put
```

# Run integration tests
```sh
mvn test -Pit
```

# Run the microservice
 --- 
```sh
mvn spring-boot:run
```

## Acces to endpoints
http://localhost:8080/cicd/api/todos
